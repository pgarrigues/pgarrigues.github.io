Tremblements de terre

Cette carte de chaleur a été réalisée à l’aide du plugin HeatMapWithTime de la librairie Folium.

Les données proviennent du site USGS recensant l'ensemble des tremblements de terre du monde.

Cliquez sur le bouton « PLAY » pour visualiser, mois par mois, l’ensemble des tremblements de terre de magnitude supérieure à 4.5 sur l’échelle de Richter.

Le script Python à l'origine de cette carte est disponible sur GitHub.

Potagers en Île-de-France

Cette carte interactive a été réalisée à l’aide du plugin MarkerCluster de la librairie Folium et de données provenant de la base de données TerriBio.

Cette carte recense l'ensemble des potagers de deux zones d'étude en Île-de-France : Saclay et Roissy.

Le plugin MarkerCluster permet de mettre en évidence des regroupements de potagers sur le territoire d'étude.

Le script Python à l'origine de cette carte est disponible sur GitHub.

Potagers en Île-de-France

Cette carte interactive a été réalisée à l’aide de la librairie Folium et de données provenant de la base de données TerriBio.

Cette dernière recense l'ensemble des potagers de deux zones d'étude en Île-de-France : Saclay et Roissy.

La méthode des mailles hexagonales permet de mettre en évidence des différences notables dans les concentrations de potagers dans nos zones d'études.

Le script Python à l'origine de cette carte est disponible sur GitHub.

Potagers en Île-de-France

Cette carte interactive a été réalisée à l’aide du plugin DualMap de la librairie Folium et de données provenant de la base de données TerriBio.

Cette DualMap permet de montrer l'effet de certains marqueurs socio-économiques sur la concentration et la taille des potagers sur deux zones d'études.

Pour plus d'informations, je vous invite à lire cette étude.

Le script Python à l'origine de cette carte est disponible sur GitHub.
