Tremblements de terre en 2020 (Heatmap)


---> Lien vers la carte.


Cette carte de chaleur a été réalisée à l’aide du plugin HeatMapWithTime de la librairie Folium.


Les données proviennent du site de l'USGS recensant l'ensemble des tremblements de terre du monde.


Cette carte permet de visualiser, mois par mois, l’ensemble des tremblements de terre de magnitude supérieure à 4.5 sur l’échelle de Richter.


Le script Python à l'origine de cette carte est disponible sur GitHub.

Suivi des bateaux du Vendée Globe


---> Lien vers la carte.


Cette carte représente la position des bateaux du Vendée Globe sur une trentaine de jours. Elle a été réalisée à l’aide du plugin TimestampedGeoJson de la librairie Folium.


Les données proviennent du site du Vendée Globe.


Lorsque vous cliquez sur "PAUSE", il est possible de connaitre le nom du bateau en cliquant sur ce dernier.


Le script Python à l'origine de cette carte est disponible sur GitHub.

Potagers en Île-de-France (Marker Clustering)


---> Lien vers la carte.


Cette carte interactive a été réalisée à l’aide du plugin MarkerCluster de la librairie Folium et de données provenant de la base de données TerriBio.


Cette carte recense l'ensemble des potagers de deux zones d'étude en Île-de-France : Saclay et Roissy.


Le plugin MarkerCluster permet de mettre en évidence des regroupements de potagers sur le territoire d'étude.


Le script Python à l'origine de cette carte est disponible sur GitHub.

---> Lien vers la carte.
